## Projetos em Destaque:

### [Local Tier List](https://github.com/Maie-r/Local-Tier-List)

Um criador de Tier Lists que armazena as listas localmente no seu computador.
Esse foi meu segundo projeto usando Blazor Hybrid, então tive mais facilidade, mas também aprendi muito mais sobre eventos. 
Desenvolvi esse projeto não apenas para ganhar mais experiência com Blazor, mas também porque queria uma forma mais prática de mostrar minhas tier lists para meus amigos, bastando abrir um aplicativo em vez de procurar um site ou uma captura de tela.

### [SmartAgenda](https://github.com/Maie-r/SmartAgenda_Continued)

Esta é uma continuação do projeto acadêmico que desenvolvi com [Isaque Caetano](https://github.com/IsaqueCN) e [Maria Carolina](https://github.com/MCarolineAlves).
Um aplicativo mobile para ajudar você a vencer a procrastinação, enviando lembretes constantes sobre tarefas, organizadas de uma forma que funciona bem para projetos grandes.
Nele você pode criar tarefas com título, descrição, prazo e subtarefas da mesma tarefa, formando uma árvore de tarefas que permite visualizar os objetivos específicos que precisam ser cumpridos, além de qual parte do projeto cada um pertence!
Também conta com um chat com IA capaz de gerenciar suas tarefas, permite configurar notificações personalizadas e exibe um painel com o seu progresso.
