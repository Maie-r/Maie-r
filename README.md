## Highlighted Personal Projects:

### [Local Tier List](https://github.com/Maie-r/Local-Tier-List)

A Tier List Maker that stores tierlists locally on your machine.
This was my second project using Blazor Hybrid, so I had an easier time, but learned a lot more about events. The reason I developed this was not only to gain more experience about Blazor, but also because I wanted to have an easier way to show my tier lists to my friends, just opening an app instead of looking up for a site or a screenshot.

### [SmartAgenda](https://github.com/Maie-r/SmartAgenda_Continued)

This is a continuation of the academic project I made with [Isaque Caetano](https://github.com/IsaqueCN) and [Maria Carolina](https://github.com/MCarolineAlves).
A mobile app to help you get away from procrastination, giving you constant reminders about tasks, set up in a way that works for large projects.
In it you can make tasks, with title, description, deadline, and sub-tasks of that same task, with which you can make a tree of tasks that let you know the specific objectives to be met, and which branch of the project it is from!
Also has an AI chat feature that can manage your tasks, allows for custom notifications and shows a dashboard of your progress.
